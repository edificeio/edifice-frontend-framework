# useDate

The `useDate` hook is used to compute user-friendly dates from various formats.

## Usage

```typescript
import useDate from './useDate';

const { fromNow, formatDate } = useDate();

const date = '2021-03-24T16:36:05.398';

return (
  <div>
    <p>From Now: {fromNow(date)}</p>
    <p>Formatted Date: {formatDate(date, 'long')}</p>
  </div>
);
```

## Return Value

The hook returns an object with the following properties:

- `fromNow`: A function that computes a user-friendly elapsed duration between now and a given date.
- `formatDate`: A function that formats a given date into a specified format.
