# useConversation

The `useConversation` hook is used to fetch the count of unread messages and the link to the messaging application.

## Usage

```typescript
import useConversation from './useConversation';

const { messages, msgLink, zimbraWorkflow } = useConversation();

return (
  <div>
    <p>Unread Messages: {messages}</p>
    <a href={msgLink}>Go to Messaging</a>
  </div>
);
```

## Return Value

The hook returns an object with the following properties:

- `messages`: The count of unread messages.
- `msgLink`: The link to the messaging application.
- `zimbraWorkflow`: A boolean indicating whether the Zimbra workflow is enabled.
