# useLibraryUrl

This hook searches for the Library app in the user's apps and returns the Library URL as a string. If no library is found in the user's apps, it returns null.

## Usage

```typescript
import useLibraryUrl from './useLibraryUrl';

const Component = () => {
  const libraryUrl = useLibraryUrl();

  return (
    <div>
      {libraryUrl ? (
        <a href={libraryUrl}>Go to Library</a>
      ) : (
        <p>No Library app found.</p>
      )}
    </div>
  );
};
```

## Returns

- `string | null`: The Library URL as a string if found, otherwise null.

## Example

```typescript
const libraryUrl = useLibraryUrl();
console.log(libraryUrl); // Outputs the Library URL or null
```
