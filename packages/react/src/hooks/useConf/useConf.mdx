# useConf

The `useConf` hook is used to fetch configuration data for a given application code using the `odeServices` client.

## Warning

⚠️ This hook should not be used outside of `EdificeClientProvider` and is not exported from the library.

## Usage

```typescript
import useConf from './useConf';
import { App } from '@edifice.io/client';

const { data, error, isLoading } = useConf({ appCode: App.MY_APP_CODE });

if (isLoading) {
  return <div>Loading...</div>;
}

if (error) {
  return <div>Error: {error.message}</div>;
}

return <div>Configuration: {JSON.stringify(data)}</div>;
```

## Parameters

- `appCode` (`App`): The application code for which the configuration is to be fetched.

## Return Value

The hook returns an object with the following properties:

- `data`: The configuration data.
- `error`: Any error that occurred during the fetch.
- `isLoading`: A boolean indicating whether the fetch is in progress.
