# Views Icons

To use this set of icons, import `@edifice-ui/icons/views` into your code:

- `import { Parent, Student, Teacher } from "@edifice-ui/icons/views;"`

## Usage

Then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <div>
      <Parent />
    </div>
  );
}
```
