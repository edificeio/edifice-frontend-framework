# Common Icons

To use this set of icons, import the `Icon` component from `@edifice-ui/react` into your code:

- `import { Icon } from "@edifice-ui/react;"`

## Usage

Then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <button>
      <Icon name="add-user" /> Add a new user
    </button>
  );
}
```
