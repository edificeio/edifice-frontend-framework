# Reactions Icons

To use this set of icons, import `@edifice-ui/icons/reactions` into your code:

- `import { Reaction1, Reaction1Circle } from "@edifice-ui/icons/reactions;"`

## Usage

Then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <div>
      <Reaction1 />
    </div>
  );
}
```
