# Iconography

- All icons are provided as a SVG Spritesheet
- All icons are sorted by category in the Spritesheet
- All icons are single color (currentColor)
- All icons has square canvas (Default to 24x24)
- All icons are for decorative purposes

## Common Interface Icons

<Alert type="info">You can copy the name of the icon by clicking on it</Alert>

To use this set of icons, import `Icon` component from `@edifice.io/react` into your code:

```jsx
import { Icon } from '@edifice.io/react';

<Icon name="<name>" />;
```

<div className="grid-css">
  {iconNames
    .filter(
      (name) =>
        !name.startsWith('apps/') &&
        !name.startsWith('nav/') &&
        !name.startsWith('audience/'),
    )
    .sort()
    .map((name) => {
      return (
        <div
          className="grid-item"
          onClick={() => {
            navigator.clipboard.writeText(name);
          }}
        >
          <Icon name={name} />
          <p>{name}</p>
        </div>
      );
    })}
</div>
