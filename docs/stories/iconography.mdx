# Iconography

- All icons are provided as a SVG Spritesheet
- All icons are sorted by category in the Spritesheet
- All icons are single color (currentColor)
- All icons has square canvas (Default to 24x24)
- All icons are for decorative purposes

## Common Interface Icons

To use this set of icons, import `Icon` component from `@edifice-ui/react` into your code:

```jsx
import { Icon } from '@edifice-ui/react';

<Icon name="" />;
```

<IconGallery>
  {iconNames
    .filter(
      (name) =>
        !name.startsWith('apps/') &&
        !name.startsWith('nav/') &&
        !name.startsWith('audience/'),
    )
    .sort()
    .map((name) => {
      return (
        <IconItem name={name}>
          <Icon name={name} />
        </IconItem>
      );
    })}
</IconGallery>

## Navbar Icons

To use this set of icons, import `Icon` component from `@edifice-ui/react` into your code:

```jsx

```

