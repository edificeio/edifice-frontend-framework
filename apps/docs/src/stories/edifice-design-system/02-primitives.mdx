# Primitives

Les primitives sont les valeurs de base du design system. Elles sont émises dans `:root` et disponibles globalement sous le préfixe `--edifice-`.

Deux fichiers sources :
- `_primitives.scss` — palette universelle, partagée par tous les thèmes
- `_primitives-legacy.scss` — palettes spécifiques aux thèmes neo et one (couleurs historiques)
