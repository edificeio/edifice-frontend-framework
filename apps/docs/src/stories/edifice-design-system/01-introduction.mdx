# Edifice Design System

Le système de thèmes d'Edifice repose sur une architecture en trois couches qui sépare clairement les valeurs primitives, les décisions de design par thème, et les surcharges par composant.

## Pourquoi ce système ?

L'ancienne approche déclarait les couleurs et la typographie directement en variables Sass Bootstrap (`$primary`, `$font-family-sans-serif`…). Ces variables étaient résolues à la compilation — impossible de changer de thème à l'exécution.

Le nouveau système génère des **CSS custom properties** scoped par `data-product`. Changer de thème revient à changer un attribut HTML : les valeurs changent dynamiquement sans recompilation.
