# Apps and Connectors Icons

This set of icons uses under the hood apps icons coming from the icons module.

```typescript
import * as IconSprites from '../../modules/icons/components/apps';
```

## Usage

Use this set of icons with the `AppIcon` Component.

```jsx
import { AppIcon } from '@edifice.io/react';

export default function App() {
  return (
    <AppIcon
      app={{
        address: '/blog',
        icon: 'blog-large',
        name: 'Blog',
        scope: [],
        display: false,
        displayName: '',
        isExternal: false,
      }}
      size="48"
    />
  );
}
```
